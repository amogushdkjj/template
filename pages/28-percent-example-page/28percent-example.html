Blue guy's bizarre adventure

Game development project for App Academy senior class.

Description

The game I built for this project, "Blue guy's bizarre adventure" is an adventure/fighting game made using the Godot game engine. In the game you can fight enemies and a boss using different attacks, collect coins, and travel between stages. The time spent for this project was about 2 months in class.




Process

A week was spent coming up with the idea and concepts for the game, I used a storyboards app to create ideas for scenes, characters, and items. I spent two weeks drawing all of the art used in the game on a digital pixel drawing program.

Rest of the time was for programming the game in Godot. This included making animations work, making attacks deduct health from enemies or the player, coding the boss battle AI, etc.

(Coding the boss character AI was probably the hardest part because I had to figure out how to make it chase and attack the player on its own)
