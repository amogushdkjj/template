Creative Labs Website

10th grade App Academy project

Description

This is a website that me and the group built for the Creative Labs club at Pasadena High School for the 10th grade final project. Creative Labs is a club to help students who want a career in game, web development, or design. They wanted a simplistic website with white space that was easy to navigate.




Process

We came up with prototypes for the website to create the best possible design. We also collected data through user testing to see how efficient the design of the website is and if people liked the look of it. Then we made changes based on the data collected and then presented the final product in a presentation.

Each person in the group was assigned a specific role and mine was primary programmer which meant I worked on coding the functionality of the website.
